# 🌟 Dreamland Shop

Bem-vindo à **Dreamland Shop**, o destino perfeito para quem busca produtos únicos, fofos e cheios de personalidade. Nossa loja oferece desde camisetas e moletons até pelúcias exclusivas, tudo pensado para espalhar diversão e o brilho de Dreamland!

---

## ✨ Funcionalidades da Interface

A interface foi projetada para ser intuitiva e acessível, contando com:

* **Menu Responsivo:** Navegação simplificada para facilitar a exploração entre as seções de Início, Produtos, Contato e Sobre Nós.
* **Acessibilidade (Controle de Fonte):** Botões dedicados para aumentar ou diminuir o tamanho da fonte, garantindo uma melhor experiência de leitura para todos os usuários.
* **Carrinho Dinâmico:** Indicador visual de quantidade de itens no carrinho de compras.
* **Seção de Destaque:** Card de apresentação de produtos com link direto para a loja.
* **Rodapé Informativo:** Links diretos para redes sociais e informações de direitos autorais.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estrutura semântica (header, nav, main, section, footer).
* **CSS3**: Estilização moderna e layout responsivo (via `main.css`).
* **JavaScript**: Lógica de interatividade para o menu e ajustes de acessibilidade.
* **Boxicons**: Biblioteca de ícones vetoriais para uma interface visualmente atraente.

---

## 📂 Estrutura de Diretórios

O projeto segue uma organização limpa de arquivos:

```text
├── assets/
│   ├── imgs/          # Logotipos, ícones (SVG) e imagens de produtos
│   ├── styles/        # Arquivos de estilização (main.css)
│   └── src/           # Scripts de comportamento (script.js)
├── index.html         # Página principal (Landing Page)
├── produtos.html      # Catálogo de produtos
├── contato.html       # Formulário de atendimento
└── sobre.html         # História da loja
```

## 🚀 Como Visualizar

1.  Clone este repositório ou baixe os arquivos.
2.  Abra o arquivo `index.html` no seu navegador favorito.
3.  Explore a interface e teste os botões de ajuste de fonte!

## 🎨 Design e Identidade

O projeto utiliza o ícone de estrela (`icon-star.svg`) como símbolo principal, remetendo ao universo mágico de Dreamland. O design busca equilibrar a estética geek com a funcionalidade de uma loja virtual moderna.
